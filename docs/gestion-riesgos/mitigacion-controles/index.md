# Unidad: Mitigación y Control de Riesgos

> **Asignatura:** Gestión de Riesgos y Vulnerabilidades | **Horas:** 2 hrs

Esta unidad cubre los tipos de controles de seguridad disponibles para mitigar riesgos identificados, las buenas prácticas de la industria, y la elaboración de reportes técnicos de vulnerabilidades en el formato adecuado para la cadena de comando militar.

## Contenido de esta unidad

- Controles, Buenas Prácticas y Reporte Técnico — 1 hr
- Examen — Gestión de Riesgos y Vulnerabilidades — 1 hr
