# Unidad: En la Red

> **Asignatura:** Seguridad de la Información y Criptografía | **Horas:** 12 hrs

Esta unidad aborda la protección de la red como segunda capa defensiva. Se cubren el control de tráfico web mediante Proxy Squid, la detección y prevención de intrusiones con Suricata (IDS/IPS), y el despliegue de honey pots para capturar actividad maliciosa y generar inteligencia.

## Contenido de esta unidad

- Proxy Squid — Control de Tráfico Web (Parte 1) — 1 hr
- Proxy Squid — Configuración y ACLs (Parte 2) — 3 hrs
- IDS e IPS — Sistemas de Detección y Prevención (Parte 1) — 1 hr
- IDS e IPS — Suricata en Práctica (Parte 2) — 3 hrs
- Honey Pots — Entornos de Engaño (Parte 1) — 1 hr
- Honey Pots — Despliegue y Análisis (Parte 2) — 3 hrs
- Examen Final — 1 hr
